# Finote

Uma maneira simples de gerenciar suas finanças pessoais!

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/GabriPires/finote.git
```

Entre no diretório do projeto

```bash
  cd finote
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```

## Stack utilizada

**Front-end:** React, Next.js, TailwindCSS

- [ ] Cadastrar usuário
